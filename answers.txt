Q1: Why might we consider using Git?

Answer: It is very useful for version control and maintaining code organization. It allows for management of code and reduces confusion, or the likelihood of you losing the version of code that you are working on. Moreover, it makes collaboration much easier because you can push and pull from a repo. 


Q2: What field could data science be applied to? Give an Example.

Answer: Data science could be applied to many fields. One such example is the medical field, as data about patients can be analyzed to glean information about cause or spread of disease.


Q3: What is the difference between Merging and Rebasing?

Answer: Merging creates a new commit, while Rebasing rewrites commit history by applying the changes from one branch to another.
